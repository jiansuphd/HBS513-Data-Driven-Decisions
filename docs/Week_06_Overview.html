Regression Discontinuity Lab

Overview

This lab in Week 6 introduces your first quasi-experimental method: Regression Discontinuity. You’ll learn how to estimate a treatment effect at a cutoff and evaluate whether an observable jump in outcomes can be credibly interpreted as causal. You will implement RD using structured design elements and core commands in Stata.

Learning Objectives

At the end of this week, you will be able to:

  • Estimate an RD model in Stata
  • Compute and interpret treatment effects at a cutoff
  • Visually inspect discontinuities using plots
