Difference-in-Differences (DiD) Lab

Overview

Welcome to Difference-in-Differences! This lab will walk you through estimating the causal effect of a policy by comparing treated and comparison groups before and after intervention. You’ll learn to run DiD regressions in Stata, create diagnostic graphs.

Learning Objectives

At the end of this week, you will be able to:

  • Set up and run DiD models using Stata
  • Interpret the DiD interaction term
  • Visualize treatment vs. comparison trends over time
  • Explain the importance of parallel trends
